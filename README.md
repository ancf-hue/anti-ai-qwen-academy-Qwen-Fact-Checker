🏛️ anti-ai-qwen-academy-Qwen Fact-Checker
Офлайн-курс по верификации ИИ-контента от Qwen • 100% бесплатный • Без сбора данных

📋 Описание
Qwen Fact-Checker — это бесплатный офлайн-курс для освоения профессии AI-верификатора (контролёра качества ИИ-контента). Приложение обучает находить «галлюцинации» ИИ, проверять факты, выявлять дипфейки и работать с регуляторными требованиями.

🎓 После прохождения курса вы сможете:
Распознавать вымышленные факты и логические ошибки в ответах ИИ
Проводить перекрёстную проверку источников и цитат
Оценивать соответствие контента гайдлайнам бренда и законодательству
Работать с инструментами детекции дипфейков и проверки метаданных
Оформлять портфолио верификатора для трудоустройства

🏆 Сертификат: При успешной сдаче финального экзамена (10/10) вы получаете именной сертификат с уникальным ID, который можно скачать, распечатать или импортировать.

✨ Особенности
🔐 Полная приватность
100% офлайн: Не требует интернета после загрузки
Локальное хранение: Все данные (прогресс, сертификат) сохраняются только в вашем браузере (localStorage)
Нет телеметрии: Приложение не собирает и не передаёт никакие данные

🌐 Двухязычный интерфейс
Мгновенное переключение между русским и английским языками
Полная локализация всех элементов: уроки, задания, сертификат, справка

🔊 Озвучка (Text-to-Speech)
Поддержка голосового сопровождения уроков и подсказок
Автоматический выбор голоса под язык интерфейса
Работает через встроенный Web Speech API браузера

🎓 Интерактивное обучение
10 уроков с практическими заданиями и автоматической проверкой
Финальный экзамен из 10 вопросов (требуется 100% результат)
Подсказки и эталонные ответы для самопроверки

🛡️ Практика и библиотека

База знаний: чек-листы, SOP, инструменты верификации

Пройдите уроки последовательно (следующий разблокируется после завершения текущего)

Пройдите все 10 уроков

Скачайте, распечатайте или проверьте подлинность в блоке валидации
